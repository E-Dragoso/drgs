カルドセプト エキスパンション　＞　カード採用基準

クリーチャーは基本的に次の特性を満たしていれば採用に値すると考えます。
  • 80G以下（多くの土地は80Gのため）
  • 領地コストなし
  • HP50↑ or 防御向けの特性持ち
アイテムは無効化系、スペルは移動系の優先度が高いです。
-----
神
引けたら絶対採用、記録はこいつらにかかっている。
  • ケルピー R
    • 水属性領地に配置すると敵セプターに足止め効果。HP30コスト85G+水領地という採用基準的には完全不合格なスペックですが特性が凶悪。 シリーズを通して最強のNPC潰しクリーチャーですが、Exでは足止め効果をNPCが考慮しないため分岐回避もされにくく強さも数段上。
  • オールドウィロウ R
    • 火属性領地に配置すると敵セプターに足止め効果。ケルピーよりHPが10高いものの領地コストが2でマップによっては厳しい。 ゼネスのブック4とバルテアスが火強打のS・ジャイアント持ちなのも少し向かい風。
Tier1
何枚でも無条件に入る最強カード群。
  • デコイ S
    • 巻物以外のダメージを反射するという戦闘では最高峰の特性持ち。 Ex時代は武器も巻物も持てるため侵略にも使え、HP20以下のため避雷針もできるという万能っぷり。 グロースボディやランドプロテクトをかければ拠点化する事もできてしまう。これって壊れてる。
  • アンシーン R
    • 反射を無効に置き換えたデコイ。おまけに素でST20。 侵略には向かないものの基本的にデコイと同じ運用ができるやはり万能クリーチャー。
  • ガスクラウド N
    • 巻物以外のダメージを半減。無効と違い落とされる可能性が出てきますがLv1火属性土地に置いた時点で実質HP80。まず死なない。 デコイやアンシーンと異なりHPが30あるためマジックボルトを撃たれにくく拠点候補の1体。 打たれ強さを生かして侵略にも使えたり。 初期ブックで火が当たりなのはこいつが入っているから。
  • バンディクート S
    • 巻物以外のダメージを1/2反射。ガスクラウドとほぼ同じだが殴り性能が高い。但し巻物は読めない。
  • ウーズ S
    • 地属性のガスクラウド。HPが40あり、Lv1地属性土地に置いたとたんに実質100。やべえ。 STが低いので流石に侵略には使えないが一応巻物が読めるため土地を奪えることも…？
  • ノーム S
    • 配置した土地の通行料が1.5倍。防御特性は持たないものの通行料アップは凶悪。 上記のクリーチャー群と比べると運用が難しいものの、ゲームスピードを上げられるため最強の一角判定。
  • グレムリン N
    • 対戦相手のアイテムを破壊。稼ぎ無しでシナリオを進めた場合NPCと明確に差が付くのがアイテムの性能だが、グレムリンはそれを破壊する事で同じ土俵に引きずり出してくる。 巻物を持たせれば侵略にも使え、手札を覚える必要が無くなるので脳内メモリーにも優しい。但し土地Lvが低いとアイテム無しでも殴り殺されるスペックなのでそこだけは注意。 初期ブックで風が大当たりなのはこいつが入っていて他も強いから。
  • ミルメコレオ R
    • 死亡時にドラゴンフライとして復活。確実に1回は土地を守れるので絶対に通行料を取れる鬼。 但しドラゴンフライは避雷針になってしまうので復活後は即座に交換等しないと危険。
  • カウンターアムル S
    • 使用時に巻物以外反射を付与。デコイ化。 道具カテゴリのため使用制限に引っかからない。ちょっとお高いものの極めて強力。
  • ガセアスフォーム N
    • 使用時に巻物以外無効を付与。アンシーン化。 カウンターアムルよりちょっとお安く、レアリティも低いので出やすい。
  • ネクロスカラベ S
    • 死亡時にゾンビとして復活。 異常にコストが低い上にゾンビ化した直後は毒を貰わないためHP50のクリーチャーが場に残る。 どの道交換必須だが即座にマジボされないのでミルメコレオより少しだけ安心度が高い。
  • ペトリフストーン S
    • HPを80に固定。無敵ではないもののほぼあらゆる攻撃に耐えられる。 アムルガセアスと比べると巻物にも耐性があるが巻物はほぼ考慮しなくていいのでメリットは小さい。
  • ホーリーグレイル S
    • 巻物も含めて完全に無効化。代償としてGが消費されますがアイテムコスト込みでも200Gがせいぜい。 高額領地を確実に守れるなら安いものです。
  • グロースボディ N
    • 対象クリーチャーのHP+10。30組を底上げしたり20組をマジボ圏内から離脱させたり。 ただしマジボの優先度が上がるので元々HPの高いクリーチャーに使うのは避けた方が良い事も。
  • テレキネシス R
    • 対象クリーチャーを隣の領地に移動。防御型だろうとお構いなしにどかせるので高額領地を無血開城させられる。 移動先に別セプターの領地があると戦闘になり、それを奪わせたり逆に地形効果を奪って返り討ちにさせたりとやりたい放題。
  • ドレインマジック S
    • 対象敵セプターの魔力を30％吸い取る。妨害だけではなく自身にもメリットのあるスペルのため有用。 周回ボーナス直後に撃ち込んだりすると楽しい事に。
  • フライ N
    • そのターンだけダイスを2個振れる。単純に移動手段として強力であり、自身に付けられた呪いを消すのにも使える。
  • ヘイスト N
    • 3ラウンド間ダイスの出目が6～8に。最強の移動スペル。
  • ホーリーワード N（0のみS）
    • 対象セプターの次のダイスの出目を固定。 基本的に相手を高額領地に嵌める目的で使うが1～3は連鎖作りに、6とXは自身の加速手段にも使える。 0は通行料の二重徴収がメインで自分に使う場面がほぼ無いので少し評価が落ちる。
  • マジックダイス R
    • 3ラウンド間ダイスの出目が5か6に。暗黒の奈落以外では安定した加速手段として使える。
  • マナ N
    • 50 × 周回数のG獲得。どんな時に使っても無駄にならない良カード。 但しExでは10G必要なので本当にGに困っている時に限って使えなかったり。
  • ランドプロテクト N
    • 対象のクリーチャーをスペルの対象に選べなくする。 ゴザ以降は基本的にマジックボルトが飛んでくるようになるためこれがあると不意の事故が減らせる。 ランプロ込みならほぼ無敵になるクリーチャーも多く非常に有用。
  • リンカネーション S
    • 手札をすべて捨てて6枚引き直す。捨てた枚数が何枚だろうと6枚引き直せる。 オーキドはかせとも呼ばれるがカルドセプトの世界ではデッキ切れが存在しないため弱点がない。 …まあケルピーを引きたいからリンカネ使いたいけど1枚しか入っていないガセアスがもう手札にある、みたいな事もままあるんですが。
Tier2
使いどころを選ぶが強力なカード群
  • イサークイーン R
    • HP=相手の持つ領地数。ゲーム終盤には鉄壁になるが序盤はイマイチなのとNPCは安い土地から手放すため相手が高額領地を踏んだ後だと悲惨な事になりがち。
  • シェイド S
    • ST40の先制持ち。HP20なので避雷針にもなるバラまき要員。
  • スチームギア S / キングトータス N / ドワーフ N
    • 非防御型でHP50かつ領地コストと配置制限なし、コスト80未満のクリーチャー群。
  • ニンジャ S
    • ST40の先制持ち。巻物強打も持っているため侵略要員になれるがちょっとコストが高い。
  • バンディット S
    • 与ダメ×2の魔力ドレイン＆援護持ち。HP20なので避雷針にもなれる。
  • リトルグレイ R
    • 50%で相手を別の土地送りにする。どうにもならない時の最終手段。
  • ウィルオウィスプ S
    • 非防御型でHP50かつ領地コストと配置制限なし。コストが85Gだが魔力ドレイン持ちのため壁に横付けして使える。
  • ケットシー S
    • ST40↑の攻撃無効＆ランプロ効果。 リリスなどのST可変系や強力な武器持ちの低STは防げない。グロースボディもかけられない。
  • アンダイン N
    • 戦闘中HPが水領地×20。水ブックなら大抵60↑が狙えるので強力。 逆に水ブック以外だと水土地をあまりとらないので微妙。ナバトもベルカイルと取り合いになるので辛い。
  • ブラッドプリン S
    • 援護に使ったクリーチャーのHPが加算される。本体性能は極めて低いので援護できずに落とされたりマジボ撃たれたりすると悲しい。
  • ムジナ S
    • HP/STが相手の手札数×10。NPCは手札を貯め込みがちなので武器を使われなければ50/50に。 ただやはり相手に依存するので安定しない。コストも安めでばらまき向き。
  • リリス N
    • STが自身の手札数×10。対戦用のガチブックならともかくシナリオで使う初期ブックに毛が生えた程度の内容では手札は溜め込まれがちなので強力。 防具こそ使えないもののリザードマンの上位種と考えて問題ない。
  • アルマジロ S
    • HPが60だったり30だったりする。30だと10G高いタイガービートルだが60だと格安のマントラップといったところ。 HPが変動した際にダメージが消えるのがちょっと便利。DSと違ってマイナスにはならないので自爆コンボはできない。
  • オルメクヘッド R
    • あらゆる攻撃を無効化し戦闘終了時にHP-10。MHPは減らないので回復可能。 Tier1の風格だが戦闘した時のHP減少は防げないためマジボ後に侵略されると確実に落ちる。 拠点とする場合は避雷針やランドプロテクトなど別途防御手段が欲しく、単体の完結力でTier1に劣ると考えこの位置に。
  • ディー・ダム R
    • 10の位が奇数の攻撃を無効化。HP40のためLv1土地に置くと50ダメージで落ちるはずだが50は無効化されるため実質HP60。Lv3なら80。計算がめんどくさい（笑）
  • ドリアード N
    • エリア内の地属性の空き地にワープできる。風以外の土地ならどこからでも連鎖のための土地取りができるのは強い。 援護も持っているので戦闘面でも強い。
  • マミー N
    • 自壊時に周回数×50G。倒されても困らない…というかむしろ終盤は倒された方がありがたい場合さえある。
  • ジーニー R
    • HP無限。即死耐性は持たないものの不死即死を使ってくる相手はシナリオにはいないため実質無敵。 高額領地には無駄侵略をしてくるゼネスに願いを減らされがち＆風領地2のコストが重いため召喚のハードルが高いのがTier1に入れない理由。
  • シムルグ S
    • 戦闘中HPが風領地×20。風属性版アンダイン。 弱点もアンダインと同様。
  • スペクター S
    • 戦闘中HP/STが10～70でランダムに変動。コストが安く死んでも困らないばらまき要員。
  • ナイトメア S
    • 貫通持ち。相手に装備が無ければ強力だが大抵ブーメランとかを投げつけられて先に死ぬ。
  • ワイバーン S
    • エリア内の風属性の空き地にワープできる。逆ドリアード。 但しコストが高く、竜族のため武器も使えない。ST40の先制持ちのため素のスペックは高い。
  • オーディンランス R / クレイモア S / マサムネ R
    • ST+50。流石に強いので使いたい。
  • カタパルト R / クロスボウ R
    • HP+20できる道具と武器。拠点向きクリーチャーは防具が使えない不死族が多くこれらのHPを増やせる装備は貴重。
  • グレムリンアムル N / ザ　ハンド R
    • 相手のアイテムを潰せるアイテム。グレムリンの項で書いた通りNPCとの一番の差は装備の性能差なのでそれを無くせるのは有難い。
  • スリング N / ファルコンソード R
    • 先制を付与できる道具と武器。防御にも侵略にも便利でコストも安め。
  • フュージョン R
    • 50の巻物ダメージ。グレムリンと組み合わせるとシナリオ中のほとんどのクリーチャーを倒せるようになる。 NPCはアイテムを貯め込みがちなため他のクリーチャーに持たせてもHPを増やされて倒しきれないことがしばしばあるのが難点。コストも高い。
  • ヘルブレイズ R
    • 40の巻物ダメージ。人族に強打するためS・ジャイアントも倒せるが基本的にサイズダウンしたフュージョン。
  • ポールスター R
    • 巻物以外を無効化するが戦闘終了時にHPが1に。マジボを引き寄せてしまうので拠点防衛としては他に劣る。
  • アップヒーバル/ウェザリング/シンク/ワイルドグロース R
    • 地形変化スペル。Gさえあればどこからでも使える上に相手の領地にも使えてしまう。 連鎖を増やすのが主な使い道。最悪相手の連鎖切りにも使えるがなるだけ自分の領地を増やす目的で使いたい。
  • アンサモン R
    • 対象のクリーチャーを手札に戻す。空いたところに移動して丸ごといただける。 序盤に引くと紙切れになってしまうのと気軽には使えないコストが難点。
  • イビルブラスト R
    • 対象のクリーチャーに30ダメージ。ガスクラウドなどHP30の強力な拠点クリーチャーをワンパンできる。 アンサモンほどではないがクリーチャー除去として信頼できる性能。
  • カモフラージュ S
    • 地形効果付与。侵略して取った土地にかけたり連鎖用にばらまいたクリーチャーにかけたり。
  • パーミッション N
    • 全ての砦を通過したことになる。Exには砦ボーナスがないため分岐ですぐ城に戻れるマップでは超強力。 逆に分岐を使っても城に戻るのに時間がかかるゴザ、分岐がそもそもないタリオやギルマンでは屁の役にも立たない。
  • バインドミスト N
    • 対象のクリーチャーに麻痺。能力とアイテム使用を縛れるので土地が取りやすくなる。 ジーニーに付けると願いが減らなくなるのでほぼ完全無敵にできるという小技も。
  • ファンタズム S
    • シナリオだとランプロとほぼ同じ。ただコストは倍以上する。
  • ホープ S
    • カードを2枚ドロー。強いけど手札が余りがちな環境ではディスカードが増えて思考への負担が大きい（笑）
  • ミューテーション S
    • MHP+20と毒付与。MHPは伸ばし過ぎるとマジボを寄せてしまったり毒ダメージで逆に死にやすくなったりするのでやや運用に注意が必要。
  • ランドトランス S
    • 土地を手放して魔力に還元。Exでは地価そのまま貰えるため確実に投資した以上の魔力を得られる。 3連鎖以上組んでレベルアップ→上げた土地をトランス→残りの土地に再投資するとより高レベルの2連鎖土地が作れる。 2連鎖をトランスすると元のレベル以上に上げられず、3連鎖以上は中々組み辛いのが難点。
Tier3
他に入れるものが無ければ
  • サイクロプス N / ジオファーグ S / G・イール N / G・スラッグ N / クンフーモンク S
    • HP50のばらまきクリーチャー…にするにはコストがちょっと高い。性能は悪くない。
  • ドレインローパー N / ピラーフレイム N / アイスウォール N / ディープスポーン S / グリーンモールド N / ストーンウォール N / ハリケーン N
    • HP50のばらまきクリーチャーでコストも80未満…だけど防御型ゆえ移動ができない。配置制限持ちも多くそもそもばらまけない事も。
  • ミノタウロス N / ワーボア N / エイドロン N
    • 色付き（ほぼ）ファイター。属性の水増ししたい時に。
  • ヘルハウンド N
    • 安価なナイト。打点は十分だが水弱打が厳しい。
  • G・アメーバ N
    • 無効が刺されば強いがそうじゃなければほぼマーフォーク。ホロビッツとグルベルに強いがどっちにもゼネスがいるので無双はできない。
  • リザードマン/ハーピー N
    • 30/30先制持ち。置かれていると厄介だが自分で拠点にするには少し頼りない。
  • トキシンフラワー S
    • ST30の先制持ち。毒も付けられるため防御型の割に殺意が高い。 ただし毒で相手を倒してもこちらがやられると土地は守れない。
  • バトルアックス S / マジックベルト S
    • ST+40。メイスやロングソードから置き換えるなら。
  • スケールアーマー S / プレートメイル R
    • HP+40/50。レザーアーマーから置き換えるなら。ただし拠点クリーチャーは不死が多く腐りがち。
  • ブーメラン S
    • HP+20ST+10＆先制。強いのだがコストが高杉晋作。半額のスリングやファルコンソードを使いたい。
  • シャッター N
    • クリーチャーでない手札カードを1枚破壊。対人では強力なカードだがNPCはスペルの枚数が少なく武器は溜め込むため効果が薄くなりがち。 暗黒の奈落でバルテアスのパーミッションやドレインマジックを割るのがメイン。相手依存かつ使い道が後ろ向きなので評価は下がってしまう。

カルドセプト エキスパンション
