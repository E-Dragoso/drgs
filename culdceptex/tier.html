カルドセプト エキスパンション　＞　カード採用基準

クリーチャーは基本的に次の特性を満たしていれば採用に値すると考えます。
  • 80G以下（多くの土地は80Gのため）
  • 領地コストなし
  • HP50↑ or 防御向けの能力持ち
アイテムは無効化系、スペルは移動系の優先度が高いです。

神
引けたら絶対採用、記録はこいつらにかかっている。
カード名 解説
ケルピー R HPはたったの30でコストも85G+水領地という残念スペックですが「水領地に配置したときの足止め効果」だけで最強クリーチャー確定。 シリーズを通して屈指のNPC潰しクリーチャーですが、初代では足止め効果をNPCが考慮しないため分岐回避もされにくく強さも数段上。 ただし前述の通り本体スペックは低いので考え無しに置くとあっさり奪われることも。もし引けたらありったけの防具をブックに詰めよう。
オールドウィロウ R 火属性版のケルピー。ケルピーよりHPが10高いのは魅力ですが領地コスト2が重い。ゼネスのブック4とバルテアスが火強打のS・ジャイアントを使ってくるのも少し向かい風。とはいえ間違いなく最高ランク。
Tier1
何枚でも欲しい最強カード群。クリーチャーはばら撒きから拠点防衛までできる対応力、アイテムは無敵系、スペルはいつ引いても使える万能のものを評価。
カード名 解説
デコイ S 巻物以外のダメージを反射するという戦闘では最高峰の能力持ち。 Ex時代は武器も巻物も持てるため侵略にも使え、HP20以下のため避雷針もできるという万能っぷり。 グロースボディやランドプロテクトをかければ拠点化する事もできてしまう。これって壊れてる。
アンシーン R 反射を無効に置き換えたデコイ。侵略には向かないものの基本的にデコイと同じ運用ができるやはり万能クリーチャー。ちなみにデコイより能力で劣る(?)のにレアリティは上。ST20の差か。
ガスクラウド N 巻物以外のダメージを半減。完全無効ではないもののLv1火属性土地に置いた時点で実質HP80。まず死なない。 またデコイやアンシーンと異なりHPが30あるためマジックボルトを撃たれにくく本命の拠点候補の1体。 初期ブックで火が当たりなのはこいつが入っているから。
バンディクート S 巻物以外のダメージを1/2反射。防御面はガスクラウドとほぼ同じですがより高い殴り性能を持ち防具も着られます。但し巻物は読めないので注意。
ウーズ S 地属性版のガスクラウド。HPが40あり、Lv1地属性土地に置いたとたんに実質100。やべえ。STが低いので流石に侵略には使えない、と思いきや一応巻物が読めるため土地を奪えることも……？
ミルメコレオ R 死亡時にドラゴンフライとして復活。確実に1回は土地を守れるというとんでもない能力。 但しドラゴンフライは避雷針になってしまうので復活後は即座に交換等しないと危険。またマジボ等で焼かれた場合は能力が発動しないのも注意。
カウンターアムル S 使用時に巻物以外反射を付与。要はクリーチャーをデコイ化するアイテム。 道具カテゴリのためどのクリーチャーも使用できる。ちょっとお高いものの極めて強力。
ガセアスフォーム N 使用時に巻物以外無効を付与。こちらはアンシーン化。 カウンターアムルよりちょっとお安く、レアリティも低いので出やすい。
ネクロスカラベ S 死亡時にゾンビとして復活。 異常にコストが低い上にゾンビ化した直後は衰弱が起きないためHP50のクリーチャーが場に残る。 即座にマジボで落とされないのでミルメコレオのドラゴンフライより少しだけ安心度が高い、がどちらにせよ続けての防衛に向かないので交換は必須です。
ホーリーグレイル S 巻物も含めて完全に無効化。代償としてGが消費されますがアイテムコスト込みでも200Gがせいぜい。 高額領地を確実に守れるなら安いものです。逆に安い土地の防衛には勿体なさ過ぎる。
テレキネシス R 対象クリーチャーを隣の領地に移動。防御型だろうとお構いなしにどかせるので高額領地を無血開城させられます。 移動先に別セプターの領地があると戦闘になり、それを奪わせたり逆に地形効果を奪って返り討ちにさせたりとやりたい放題。
ドレインマジック S 対象敵セプターの魔力を30％吸い取る。初代では貴重な増収系スペル。周回ボーナス直後に撃ち込んだりすると楽しい事に。 使うタイミングを考える必要がありマナほどの万能感はありませんが、自分を伸ばしつつオマケで妨害できるのも〇。
ヘイスト N 3ラウンド間ダイスの出目が6～8に。カルドセプト史上最強の移動スペル。
ホーリーワードX N 次の出目を10に固定。ヘイストが強過ぎて霞みますが十分過ぎるくらい強い。
マナ N 50 × 周回数のG獲得。どんな時に使っても無駄にならない良カード。 但しExではこれの使用にも10G必要なので本当にGに困っている時には使えなかったり。
ランドトランス S 土地を手放して魔力に還元。初代では地価の満額貰えるのでゴブリンを置いて即トランスしても土地代-20G貰えます。 3連鎖以上組んでレベルアップ→上げた土地をトランス→残りの土地に再投資するのが基本。これができると護符投資も不要に。マップによっては3連鎖以上を組むのが難しいのが難点か。
リンカネーション S 手札をすべて捨てて6枚引き直す。捨てた枚数が何枚だろうと6枚引き直せる。 ポケカのオーキドはかせにも例えられますがカルドセプトの世界ではデッキ切れがないためノーリスク。 ……まあケルピーを引きたいからリンカネ使いたいけど1枚しか入っていないガセアスがもう手札にある、みたいな事もままあるんですが。
Tier2
Tier1ほどではないけど強力なカード群。大抵の場合は最後までブックに入ります。
カード名 解説
イサークイーン R HP=相手の持つ領地数×10。ゲーム終盤には鉄壁。序盤はイマイチなのとNPCが高額領地を踏んだ後だと悲惨な事になりがち。
シェイド S ST40の先制持ち。HP20なので避雷針にもなるばら撒き要員。
スチームギア S HP/ST50でどこにでも置け、コストも80未満の超優良クリーチャー。 こいつを置いた土地を育てる場面がまずないので通行料半額のデメリット能力もあって無いようなものです。
ニンジャ S ST40の先制持ち。巻物強打も持っているため侵略もこなせますがちょっとコストが高い。
バンディット S 与ダメ×2の魔力ドレイン＆援護持ち。HP20なので避雷針にもなれる。

キングトータス		 S
N		 非防御型でHP50かつ領地コストと配置制限なし、コスト80前後のクリーチャー。スチームギアと違って侵略はできませんが属性持ちのため拠点にできます。 ウィスプはST0に横付け移動侵略で小遣い稼ぎも可能。
ケットシー S ST40↑の攻撃無効＆ランプロ効果。かなり強いもののリリスなどのST可変系や低ST+強武器は防げず、グロースボディもかけられないのが残念。
アンダイン N 戦闘中HPが水領地×20。水ブックなら大抵60↑が狙えるので強力。水ブック以外では全く役に立たないほか、ナバトのように土地が少ないマップも厳しい。
ブラッドプリン S 援護に使ったクリーチャーのHPが加算される。本体性能は極めて低いので援護要員がいなかったりばら撒き後にマジボを撃たれたりすると悲しい。
ムジナ S HP/STが相手の手札数×10。NPCは手札を貯め込みがちなので武器を使われなければ50/50に。コストも安めでばらまき向き。 ただどうしても相手に依存するため不確定要素が残る。
リリス N STが自身の手札数×10。対戦用のガチブックならともかくシナリオで使う程度のブックでは手札はだぶつきがちなので強力。 防具こそ使えないもののリザードマンの上位種と考えて問題ない。
アルマジロ S HPが60だったり30だったりする。30になると10G高いタイガービートルですが60になれば格安。 HPが変動した際にダメージが消えるのがちょっと便利。DSと違ってマイナスにはならないのでグース自爆コンボはできません。
オルメクヘッド R あらゆる攻撃を無効化し戦闘終了時にHP-10。MHPは減らないので回復可能。 Tier1の風格ですが戦闘した時のHP減少は防げないためマジボ後に侵略されると確実に落ちます。 拠点とする場合は避雷針やランドプロテクトなど別途防御手段が欲しく、単体の完結力でTier1に劣ると考えこの位置に。でもTier1.5くらいの気がする(?)。
ディー・ダム R 10の位が奇数の攻撃を無効化。HP40のためLv1土地に置くと50ダメージで落ちる、と思いきや50ダメージを無効化できるため実質HP60。Lv3なら70を無効化して実質80。計算が若干めんどくさい（笑）。
ドリアード N エリア内の地属性の空き地にワープできる。風以外の土地ならどこからでも連鎖のための土地取りができるのは強い。 援護も持っているので戦闘面でも強い。でも地ブック以外には要らない……。
ノーム S 配置した土地の通行料が1.5倍。通行料アップはゲーム展開を早める可能性があるのでRTA的には評価が高い。アイテム込みでないと厳しい数値なのでそこがネック。そしてこいつを上げると何故か踏まれなくなるんですよね不思議なことに。
マミー N 自壊時に周回数×50G。倒されても困らない、どころか倒された方がありがたい場合さえある。 ばら撒き要員としては極上で配置制限もないのでどのブックに入れても〇。
グレムリン N 対戦相手のアイテムを破壊。中盤以降はどうしてもNPCとアイテム性能の差が開いていきますが、グレムリンはそれを破壊する事で同じ土俵に引きずり出す事ができます。 初代は巻物を読めてしまうので侵略にも使え、相手の手札を覚える必要が無くなるので脳内メモリーにも優しい。但し本体スペックがゴブリンでばら撒きには不向きなためTier1から降格。 初期ブックで風が当たりなのはこいつが入っていて他も強いから。
ジーニー R HP無限。即死耐性は持たないものの不死への即死を使ってくる相手はシナリオにはいないため実質無敵。 ゼネスの無駄侵略に願いを減らされがち＆風領地2のコストが重いためTier2止まり。
シムルグ S 戦闘中HPが風領地×20。風属性版アンダイン。弱点もアンダインと同様。
スペクター S 戦闘中HP/STが10～70でランダムに変動。意外と頑張ってくれるし戦闘が無駄に盛り上がって楽しい(？)。

シャッター N クリーチャーでない手札カードを1枚破壊。対人では強力なカードだがNPCはスペルの枚数が少なく武器を溜め込むため効果が薄くなりがち。 デコイアンシーンを守るためにスクロールを割ったり、暗黒の奈落でバルテアスのパーミッションやドレインマジックを割るのがメイン。 どちらにせよ妨害しかできないので評価は低い。
カオスパニック
バックワード		 S
N		 逆進行できるカード。稀に欲しい場面があるので少なくともキュアーよりは使える。タリオでのカオスパニックはハメ技にも使えます。
バリアー N 6R(SS版は3R)の間スペルの対象に選ばれなくなる。ほとんどのマップが3人戦で、これがかかっている間は妨害スペルが自分に飛んでこなくなるのでNPC同士で足を引っ張りあってくれるように。 妨害そのものに旨味はないものの自分が遅延を食らわなくなるためRTA的にはありがたいスペル。但しヘイストやホリワもかけられなくなりフライで打ち消されるのは注意。
その他ホーリーワード N(0のみS) 対象セプターの次のダイスの出目を固定。自分にも相手にも使えて6は自身の加速手段にも使えます。 但し大量に入れても大抵は手札で腐るだけなのでほどほどに。
